# 🚨 Sentinela de Cultivo: Monitor de Temperatura e Umidade

Um sistema de monitoramento focado em alertas críticos para estufas e ambientes controlados, utilizando a plataforma Arduino.

## 📋 Sobre o Projeto

O objetivo deste projeto é fornecer uma interface de leitura clara e um sistema de segurança confiável. Ele atua como um vigia 24/7, disparando alarmes caso o ambiente se torne hostil para o cultivo.

### 🛡️ Sistema de Segurança
- **Monitoramento:** Leitura em tempo real via sensor DHT.
- **Interface:** Display LCD 16x2 (I2C) com atualização dinâmica.
- **Alarme Sonoro:** Buzzer piezoelétrico para alertas de áudio.
- **Alerta Visual:** LED de alto brilho para sinalização de erro.

---

## 🛠️ Hardware Necessário

| Componente | Função |
| :--- | :--- |
| **Arduino Uno/Nano** | Processamento central |
| **Sensor DHT11 ou DHT22** | Coleta de Temp/Umid |
| **Display LCD 16x2 com I2C** | Interface visual dos dados |
| **Buzzer Ativo/Passivo** | Sinal sonoro de alerta |
| **LED Vermelho + Resistor** | Sinal visual de alerta |
| **Protoboard & Jumpers** | Conexões de circuito |
